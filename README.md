# Objetivo
Este projeto tem como objetivo executar microserviços em kubernetes usando Rancher Descktop, controlado por GitOps com ArgoCD apartir de um repositorio publico
## Ferramentas
- Git
- GitHub
- Rancher Descketop
- Kubernetes
- ArgoCd
- Docker
